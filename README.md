# 基于React实现的仿网易云web应用

[在线预览地址](http://youky.top/netease_music/ ) 



## 主要功能

- 首页推荐
- 歌手列表
- 歌手详情
- 排行榜
- 歌单详情
- 播放器
- 播放列表
- 搜索



## 主要技术栈

- `React1`(V17)
- `react-router`(V6)
- `redux`
- `redux-immutable`
- `redux-thunk`
- `styled-components`



### 动画库

- `animation.css`
- `react-transition-group`



### 针对移动端的改善

- `better-scroll`



### 后端接口

采用 [Binaryify](https://github.com/Binaryify)大佬提供的[在线接口](https://github.com/Binaryify/NeteaseCloudMusicApi)
